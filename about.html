About the Z-Shift Computation Tool

The Z-Shift Computation Tool is designed to help researchers and practitioners analyze credit rating migrations by computing the z-shift between observed transition matrices and a stable (neutral) matrix. This tool allows users to input their own matrices, specify the grid size (number of states), and obtain the z-shift values along with visualizations.

By making this tool accessible, we aim to advance economic science by providing an easy-to-use platform for credit risk analysis and other applications involving transition matrices.

Features:

  • Customizable grid size and number of matrices.
  • Input stable and observed matrices directly in the browser.
  • Compute z-shift values using robust algorithms.
  • Visualize results through interactive charts.
  • Export results for further analysis.

Note: All computations are performed locally in your browser. No data is sent to any server, ensuring your data remains private.
